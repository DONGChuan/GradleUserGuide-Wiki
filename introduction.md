# 介绍

很高兴能向大家介绍 Gradle,
这是一个构建系统, 
我们认为它是 java ( JVM ) 世界中构建技术的一个飞跃. 

Gradle 提供了:

* 一个像 Ant 一样的非常灵活的通用构建工具
* 一种可切换的, 像 maven 一样的基于合约构建的框架
* 支持强大的多工程构建
* 支持强大的依赖管理(基于 ApacheIvy )
* 支持已有的 maven 和 ivy 仓库
* 支持传递性依赖管理, 而不需要远程仓库或者 pom.xml 或者 ivy 配置文件
* 优先支持 Ant 式的任务和构建
* 基于 groovy 的构建脚本
* 有丰富的领域模型来描述你的构建
